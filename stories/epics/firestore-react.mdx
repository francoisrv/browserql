# Firestore with React

Implement Firestore with browserql with React

```jsx
const schema = gql`
  type Todo @firestore {
    title: String!
  }
`

<BrowserqlProvider extensions={[connectFirestore(db, schema)]}>
  <Firestoreql paginate="Todo">
    {todos => (
      <ul>
        {todos.map(todo => <li key={todo.id}>{todo.title}</li>)}
      </ul>
    )}
  </Firestoreql>
</BrowserqlProvider>
```
