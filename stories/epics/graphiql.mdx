# GraphiQL

GraphiQL for browserql

```jsx
const schema = gql`
  extend type Query {
    hello: String!
  }
`

const queries = {
  hello() {
    return 'hello'
  }
}

<BrowserqlProvider schema={schema} queries={queries}>
  <GraphiQL />
</BrowserqlProvider>
```
